To-Do List App 📝
A functional, single-page To-Do List application built using core web technologies. This project allows users to manage daily tasks through a clean, responsive interface with data persistence.

🚀 Features
Task Management: Add new tasks to the list using the input field and "Add" button.

Interactive UI: Click a task to toggle a "checked" state, which visually strikes through the text and changes the bullet icon.

Persistent Storage: Uses localStorage to ensure tasks remain available even after the browser is refreshed or closed.

Task Deletion: Permanently remove tasks from the list by clicking the delete icon (\u00d7).

Modern Aesthetic: Features a linear-gradient background and custom Poppins typography.

🛠️ Tech Stack
HTML5: For the application structure.

CSS3: For styling, including linear gradients (#153677 to #4e085f) and responsive design.

JavaScript (ES6): For DOM manipulation, event delegation, and Local Storage synchronization.

📂 Project Structure
index.html — The main entry point.

style.css — Contains all visual styling and layout rules.

script.js — Handles task logic, event listeners, and data storage.

images/ — Folder containing icon.png, unchecked.png, and checked.png.

📥 Getting Started
Clone this repository to your local machine.

Ensure the images folder is in the same directory as your HTML file.

Open index.html in any modern web browser to start managing your tasks.
